# Public scope

This repository is intentionally public-safe. It is a small research harness for
forecast benchmarking, not a CrudeCode production component.

Allowed here:

- synthetic fixtures
- user-supplied CSV inputs
- standalone benchmark code
- metrics, tests, and methodology docs

Not allowed here:

- proprietary well data
- licensed/vendor data exports
- private CrudeCode database schemas or credentials
- customer, fund, or dataroom data
- production ETL or acquisition code

The goal is to make forecast claims testable without exposing private data.
